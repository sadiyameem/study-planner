# Study Planner

# Description
- This is a simple web-based study planner that helps people organize their study schedule. Users can add subjects, set goals, generate time, track completion and download their timetable as a pdf.

- # Features
 - Weekly day selector
  - Add study tasks (subject, time and goal or task)
  - Time slot generator (start time and duration)
  - Mark tasks as complete
  - Download as pdf
 
  # Tools
  - HTML
  - CSS
  - JavaScript
  - html2pdf
  - jspdf
 
  # Links
  - https://htmlcolorcodes.com/
  - https://cdnjs.com/libraries/html2pdf.js/0.8.0
  - https://cdnjs.com/libraries/jspdf
 
  # How it works
  1. Select a day
  2. Enter a subject, goal and time
  3. Use the time generator
       - Choose a start time
       - Select a duration
  4. Click Add to Timetable
  5. View tasks
  6. Mark tasks
  7. Download your week as a PDF
